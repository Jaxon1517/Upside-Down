
	-----= How To Make Levels =-----

	Image rules:
		a. Must be a bmp
		b. Must be 24x256
		c. Names must be included in the source code

	Making Levels:
		a. It is suggested to put the level number in the top left in lava.
		b. The colors
			Red = Land
			Green = Lava
			Blue = Comment
		c. The end of a level is automatically the end.
